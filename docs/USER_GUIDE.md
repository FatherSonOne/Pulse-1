## Getting Started

### Accessing Messages

1. Click **Messages** in the left sidebar
2. View your thread list on the left (30% of screen)
3. Select a thread to view conversation on the right (70%)
4. Start typing in the message input at the bottom

### Interface Overview


## Basic Messaging

### Sending Messages

1. **Click** message input at bottom
2. **Type** your message (max 2000 characters)
3. **Press** Enter or click Send button

**Keyboard Shortcuts**:
- `Ctrl/Cmd + Enter` - Send message
- `Shift + Enter` - New line

### Editing Messages

1. **Hover** over your message
2. **Click** three dots menu
3. **Select** "Edit"
4. **Make changes** and press Enter

### Deleting Messages

1. **Hover** over your message
2. **Click** three dots menu
3. **Select** "Delete"
4. **Confirm** deletion

## Reactions

### Adding Reactions (Desktop)

**Method 1: Hover**
1. **Hover** over any message for 300ms
2. **Quick reaction bar** appears above/below
3. **Click** emoji to add reaction

**Method 2: Right-click**
1. **Right-click** message
2. **Select** "Add Reaction"
3. **Choose** emoji from picker

### Adding Reactions (Mobile)

1. **Long-press** message for 500ms
2. **Reaction bar** appears
3. **Tap** emoji to add

### Removing Reactions

1. **Click** your reaction on the message
2. Reaction is removed instantly

## Threading

### Creating a Thread

1. **Hover** over message
2. **Click** reply icon
3. **Type** reply in thread view
4. **Send** to create thread

### Viewing Threads

- **Thread count** appears below original message
- **Click** "X replies" to view thread
- **Back button** returns to main conversation

## Tools & Features

### Opening Tools Panel

1. **Click** Tools button (top-right)
2. **Browse** 39 tools in 4 categories:
   - AI Tools (purple)
   - Content Creation (blue)
   - Analysis (green)
   - Utilities (amber)

### Popular Tools

**AI Coach**: Get personalized guidance
- Click "AI Coach" tool
- Describe your situation
- Receive AI-powered advice

**Templates**: Use message templates
- Click "Templates" tool
- Select template
- Customize and send

**Smart Compose**: AI suggestions
- Start typing message
- AI suggests completions
- Click suggestion to use

**Sentiment Analysis**: Check tone
- Select conversation
- Open "Sentiment Analysis"
- View tone insights

## Focus Mode

### Starting Focus Mode

1. **Click** Focus Mode button (crosshairs icon)
2. **Select** duration (5-60 minutes)
3. **Click** Start

### What Happens

- **Timer starts** counting down
- **Distractions hidden** (sidebar, notifications)
- **Only active thread** visible
- **Timer overlay** shows progress

### Pausing/Stopping

- **Pause**: Click Pause button
- **Resume**: Click Resume button
- **Stop**: Click Stop button

### After Session

- **Notification** appears
- **Focus Digest** shows missed messages
- **Break suggested** (5-15 minutes)

## Search & Organization

### Searching Messages

1. **Click** search box (top-left)
2. **Type** search query
3. **Press** Enter or Ctrl+J
4. **View** results in thread list

### Pinning Messages

1. **Hover** over message
2. **Click** pin icon
3. Message added to pinned section

### Starring Messages

1. **Right-click** message
2. **Select** "Star"
3. Access starred messages in sidebar

## AI Features

### Smart Compose

- **Automatically suggests** message completions
- **Press Tab** to accept suggestion
- **Continue typing** to ignore

### Voice Context

- **Record voice** message
- **AI extracts** action items
- **Automatic** transcription

### Translation

1. **Select** message
2. **Click** translate icon
3. **Choose** target language
4. **View** translation

### Conversation Intelligence

- **Automatic sentiment** analysis
- **Topic detection** in conversations
- **Engagement metrics** tracking
- **Response time** analysis

## Tips & Tricks

### Productivity Tips

1. **Use keyboard shortcuts** for speed
2. **Pin important** threads
3. **Use templates** for common messages
4. **Schedule messages** for later
5. **Focus Mode** for deep work

### Best Practices

- **Star important messages** for quick access
- **Use threads** to keep conversations organized
- **Add reactions** instead of short replies
- **Review Focus Digest** after each session
- **Pin frequently used tools**

### Advanced Features

- **Search with filters** (date, sender, type)
- **Export conversations** for records
- **Link to CRM** contacts
- **Set up auto-responses**
- **Create message templates**
