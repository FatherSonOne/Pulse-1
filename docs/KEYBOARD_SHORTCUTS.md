# Pulse Messages - Keyboard Shortcuts

**Version**: 1.0
**Last Updated**: 2026-01-19

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `?` | Show shortcuts help | Display this shortcuts panel |
| `Esc` | Close modal/panel | Close any open modal or panel |
| `Ctrl/Cmd + K` | Command palette | Open command palette |

---

## Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + ]` | Next thread | Move to next thread in list |
| `Ctrl/Cmd + [` | Previous thread | Move to previous thread |
| `Ctrl/Cmd + J` | Jump to search | Focus search box |
| `Ctrl/Cmd + /` | Toggle sidebar | Show/hide left sidebar |

---

## Messaging

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/Cmd + Enter` | Send message | Send current message |
| `Shift + Enter` | New line | Insert line break (don't send) |
| `Ctrl/Cmd + B` | Bold text | Make selected text bold |
| `Ctrl/Cmd + I` | Italic text | Make selected text italic |
| `Ctrl/Cmd + E` | Code format | Format as code block |
| `Ctrl/Cmd + Z` | Undo | Undo last change |
| `Ctrl/Cmd + Y` | Redo | Redo last undone change |

## Tips

### Chord Shortcuts

Some shortcuts require multiple keys in sequence:
- `G` then `H` - Go to home
- `G` then `S` - Go to starred
- `G` then `D` - Go to drafts

### Customization

Customize shortcuts in Settings > Keyboard Shortcuts

### Platform Differences

- **Windows/Linux**: Use `Ctrl`
- **Mac**: Use `Cmd` (âŒ˜)
